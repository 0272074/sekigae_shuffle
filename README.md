# 席替え案 作成ツール (Seat Shuffler)

教室の席替えを効率的に行うためのWebアプリケーションです。様々な条件を設定して、公平で適切な座席配置を自動生成できます。

![スクリーンショット](screenshot.png)

##  特徴

- **完全ローカル動作**: サーバー不要、ブラウザだけで動作
- **豊富な条件設定**: 位置指定、距離条件、班条件など多彩な制約に対応
- **バッジシステム**: 児童にバッジを付けてグループ化、配置ルールを設定
- **履歴管理**: 過去の席替え履歴を保存参照
- **履歴ベース条件**: 前回と同じ席や隣を避ける設定が可能
- **データ永続化**: ブラウザのローカルストレージに自動保存
- **インポート/エクスポート**: JSONファイルでバックアップ復元

##  使い方

### 1. ファイルを開く
`index.html` をブラウザで開くだけで使用できます。

### 2. 児童登録
- 出席番号、名前、性別を入力
- 必要に応じてバッジ（タグ）を設定

### 3. 座席配置
- 教室の縦横サイズを設定
- 各座席をクリックして種類を変更（通常/使用不可/男子のみ/女子のみ）
- 班番号を設定

### 4. 条件設定
**個別条件（児童ごと）:**
- 場所指定
- 前/後/左/右から列目まで
- 特定の児童と離す/近くにする
- 同じ班/違う班にする

**全体条件:**
- 履歴ベース: 同じ席を避ける、同じ隣を避ける
- バッジ条件: 同じバッジの児童を近く/離す

### 5. 席替え実行
- 条件を満たす配置を自動生成
- 気に入らなければ「別の案を生成」
- 座席をクリックして手動で入れ替えも可能
- 「確定履歴に保存」で名前を付けて保存

##  ファイル構成

```
seat-shuffler/
 index.html      # メインHTML
 style.css       # スタイルシート
 app.js          # アプリケーション制御
 store.js        # データ管理（localStorage）
 shuffler.js     # 席替えロジック
 views/          # 各画面のビュー
     input-view.js       # 児童登録
     layout-view.js      # 座席配置
     condition-view.js   # 条件設定
     result-view.js      # 結果表示
     student-view.js     # 発表モード
     history-view.js     # 履歴管理
```

##  技術仕様

- **フロントエンド**: HTML, CSS, JavaScript（バニラ）
- **外部依存**: なし
- **データ保存**: localStorage API
